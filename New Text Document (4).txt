# Week 01 Assignment - Registration Form

A complete registration form with validation built using HTML, CSS, and JavaScript.

## Features
- Modern gradient design
- Form validation
- Password matching check
- Email validation
- Success/error messages
- Responsive layout

## How to Use
1. Open `index.html` in any web browser
2. Fill in all fields:
   - Full Name
   - Email Address
   - Password (min 6 characters)
   - Confirm Password
3. Click "Register Now"
4. See success message with details

## Files Included
1. `index.html` - Main HTML file
2. `style.css` - Styling and design
3. `script.js` - Form validation logic
4. `README.md` - This documentation file

## Validation Rules
- All fields are required
- Email must be valid format
- Password minimum 6 characters
- Passwords must match

## Technologies Used
- HTML5
- CSS3 (with gradients and animations)
- JavaScript (ES6)
